# CommitEd - Planning & Journal de Suivi

Ce dépôt centralise les comptes-rendus de réunions et les travaux d'analyse (diagrammes, études) demandés par l'encadrant, le tout compilable via Typst.

## Structure

* **`main.typ`** : Le fichier principal qui assemble tout le rapport.
* **`/artifacts`** : Contient les **rendus d'exercices** (Diagrammes UML, études comparatives, résultats de tâches, etc).
* **`/meetings`** : Contient les comptes-rendus de chaque réunion (Journal de bord).
* **`/out`** : Dossier de destination où le PDF final existe/sera généré.

## Prérequis

**[Typst](https://github.com/typst/typst)** doit être installé sur votre machine. Sinon, vous pouvez utiliser l'éditeur en ligne sur [typst.app](https://typst.app).
